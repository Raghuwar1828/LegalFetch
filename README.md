# LegalFetch

LegalFetch is a web application that helps users extract, analyze, and understand Terms of Service and Privacy Policies from websites.

## Features

- **Extract Legal Documents**: Enter any website URL to extract its Terms of Service and Privacy Policy
- **Simplified Summaries**: Get a 100-word summary and a one-line summary of complex legal documents
- **User Management**: Create an account, sign in, and manage your profile
- **Responsive Design**: Modern UI with responsive design using CSS

## Technologies Used

- PHP
- MySQL Database
- HTML/CSS
- JavaScript
- AOS (Animate on Scroll) Library
- FontAwesome Icons

## Installation and Setup

1. Clone the repository
2. Set up a web server with PHP and MySQL
3. Import the database schema (not included in this repository)
4. Update `config.php` with your database credentials
5. Access the application through your web server

## File Structure

- `config.php` - Database connection configuration
- `index.html` - Landing page
- `signup.php` - New user registration
- `signin.php` - User login
- `home.php` - Main application interface
- `profile.php` - User profile management
- `logout.php` - User logout functionality
- `contact.php` - Contact page
- `styles.css` - Styling for the application
- `script.js` - JavaScript functionality

## Usage

1. Register for an account or sign in
2. Enter a website URL in the search box
3. Click "Generate" to extract and analyze the legal documents
4. View the Terms of Service, Privacy Policy, and summaries
